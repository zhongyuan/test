Detailed Description

template<class TestDelegate, class TestFixture>
class gaia::testing::DelegateTestCase< TestDelegate, TestFixture >

This test case provides a framework in which you can test gaia.core.Delegate classes in a controlled environment.

It provides basic support for the lifecycle of a Delegate, and hooks with which you can inject various dependencies and control the environment in which your Delegate is tested.

Inherits gaia::testing::GaiaTestCase.

TestDlelegate is extend from gaia::core::Delegate

TestFixture is extend from gtest/testing/test
