Utility method for launching an page.

The gaia::core::Demand used to launch the Page is: action = Demand#ACTION_MAIN() extras = null, unless a custom Persistence is provided here All other fields are null or empty.

NOTE: The parameter pkg must refer to the package identifier of the package hosting the page to be launched, which is specified in the gaia.xml file.

Parameters:
pkgThe package hosting the page to be launched.
pageClsThe page class to launch.
extrasOptional extra stuff to pass to the page.
