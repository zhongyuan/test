Architecture

The following figure illustrates the COS Architecuture

The COS architecture consists of following layers.

Linux Kernel

This layer contains either the real-time operating system or the Linux kernel, which provides security, memory management, process management, network and driver model and so on.

Library

This layer contains a collection of C/C++ librarys which are called by COS system components. eg libc, media, graphics, sqlite, etc.

Framework

This layer is the core of the COS system. It provides Acitivty manager, window manager, view system, content provider, package manager, telephony manager etc.

SDK and Runtime

This layer contains COS native SDK, Web runtime and java runtime, for which we can develop COS native app, web app or android app.

Application

Base on the SDK and Runtime layer, we can develop COS native app, web app and android app.
