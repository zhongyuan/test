#Install step by step#

###GAIA IDE installaction step: ###

1. Download Eclipse 3.7(Indigo) version or above from [Eclipse](http://www.eclipse.org/downloads/"Eclipse") and unzip the package to PC.

2. Download cdt-master-8.0.1.zip or above from [CDT](http://eclipse.org/cdt/downloads.php"CDT").

3. Go to your Eclipse and click on "Help->Install New Software..." in the Eclipse Menu bar.

4. Click the Add button, input an adaptable name (such as CDT), and then click the "Archive..." button. From there, choose the cdt-master-8.0.1.zip which we downloaded just now and press OK. Then, select CDT Main Features and CDT Optional Features. Click the next button twice before accepting the terms of license agreement in order to finish installation. When you have finished the steps above, please put the plug-in we provided named "com.com.gdt_1.6.7.jar" (from unizpping IDE folder) into the plugins folder in Eclipse.

6. Restart your Eclipse.
