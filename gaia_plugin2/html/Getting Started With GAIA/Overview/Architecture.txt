The GAIA architecture consists of the following layers.

##Linux Kernel##

This layer contains either the real-time operating system or the Linux kernel, which provides security, memory management, process management, network, driver model and so on.

##Library##

This layer contains a collection of C/C++ libraries called by the GAIA system components. eg libc, media, graphics, sqlite, etc.

##Framework##

This layer is the core of the GAIA system. It provides the Acitivity manager, window manager, view system, content provider, package manager, telephony manager etc.

##SDK and Runtime##

This layer contains GAIA native SDK, Web runtime and java runtime, for which we can develop GAIA native app, web app or android app.

##Application##

Based on the SDK and Runtime layer, we can develop GAIA native app, web app and android app.
