Web APIs Guide

GAIA Web Runtime supports various web APIs which enable applications handle all kinds of common tasks. Below is a brief guide on how to use these APIs.

Storage

You can use web storage APIs for basic key / value data storing.

If web storage can't meet your needs, turn to IndexedDB which provides database APIs for storing structured records and high performance searching.

Device APIs

Geolocation API: Obtain the current location of the device.

Sensors API: Obtain information given by the various sensors available on the device.

Screen orientation API: Get notification when screen orientation changes as well as control which screen orientation a page/app wants.

Vibration API: Control device vibration for things like haptic feedback in games.

Battery status API: Allow you to get battery status such as battery level and whether is in charing.

Network

The most commonly used network APIs are XMLHttpRequest. However, be sure to keep the same origin policy in mind. Your packaged application is hosted within an independent origin, so XMLHttpRequest is disabled by default for external websites, since the origins are not the same. A variant of XMLHttpRequest, named system XMLHttpRequest, is provided for cross origin communication. To use system XMLHttpRequest, you must declare "http://china-liantong.com/webruntime/api/systemXHR" feature in "config.xml", and use the following way to create the XMLHttpRequest object:

var xhr = new XMLHttpRequest({mozSystem: true});

The network information API gives you the ability to get the bandwidth information and the data connection type is being used. This is useful, for example, your application wants to choose online video's quality automatically based on network bandwidth.

The online and offline events can be used to detect whether network is connected.

A private TCPSocket API is provided for using TCP connections.

Graphics

You can use HTML5 canvas for 2D bitmap drawing and use SVG for vector graphics. You can even play with 3D graphics with WebGL.

Media

The standard HTML5 video and audio elements can be used for media playback.

GAIA Web Runtime also supports experimental camera APIs for camera preview, taking pictures and setting parameters, which allow you to develop a full featured camera application.
